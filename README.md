# Fortran-Program-For-Euler-Method
A particle of mass 10 gm and charge 0.2 C moves under electric field E = 20 V/m and gravitational field g = −9.8 m/s². At t = 0, v = −1.52 m/s and x = 7.52 m. Using Euler’s method, find position and velocity for a suitable interval until velocity becomes 1.52 m/s. Also find % error and plot position–time and velocity–time graphs.
